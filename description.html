Описание

Самый простой способ записи данных в платформу Пересвет - через команду API data/set.

Но в этом случае придётся самостоятельно заботиться о буферизации данных, о событиях изменения паратров тегов и т.д.

Всю эту работу берёт на себя коннектор - специализированная программа, умеющая забирать данные из какого-либо источника и отправлять их в платформу.

Рекомендации по созданию коннекторов:

  1. Для каждого протокола данных (Modbus, OPC и т.д.) - отдельный коннектор.

  2. Коннектор работает только с одним источником данных.
